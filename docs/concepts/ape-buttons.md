---
alias: ape-buttons
title: 'Ape Buttons'
description: 'Fast action buttons in UI'
---

Ape buttons refers to prominent, instant-action buttons.

For example, "Buy 1 SOL," "Max Long," steps like "Approve Token" using account abstraction, "Transact" that enables multi-chain transaction via chain abstraction. 

These buttons help to remove friction and help users act fast.
