# Web3 User Personas

Welcome to the Web3 User Personas project!

A showcase of [Web3 participants archetypes](personas/index.md).

## TL;DR

{==

Web3 thrives on diverse, opinionated users with unique motivations. 

*Builders, technologists, finance participants, and experimentalists coexist in
overlapping communities.*

Effective engagement requires understanding these
nuanced personas. Users embody multiple roles and interact across platforms based
on needs. Build for today, plan for tomorrow. Collaborate to keep personas accurate and evolving.

==}

## Why Web3 Personas?

User personas validate product-market fit. 

They shape value propositions and guide marketing. In Web3, personas inform critical design
 decisions. Deep, evolving persona insights drive engaging, effective user experiences.

## What is Unique?

This research offers a *Tarot-inspired spin* on Web3 user personas. It maps [[persona-archetypes|12+ user types]]
to help Web3 builders create user-centric dApps. Use this guide to prioritize impactful features and nail Web3 UX and DX.

These personas will mature and change as new users and solutions emerge. Lets collaborate to keep them alive, accurate
accessible and refined as we learn more. See [[contributions|how to get involved]] in this project.
