# Web3 User Personas

Welcome to the Web3 User Personas project!

Here is a showcase of [15 Web3 participants archetypes](personas/index.md). This documentation, built with [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

It provides detailed profiles for each persona, including their motivations, behaviors, and engagement patterns across platforms like Discord, Telegram, Twitter/X, and Reddit.

## Audience

Builders, designers creating decentralized apps and blockchain based solutions. 

Create your own user persona template using these insights. Identify key user pain points, what they need, UX and technical considerations when building for these personas.
