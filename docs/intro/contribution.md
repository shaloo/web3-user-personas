# Contributions

Web3 user personas is an open source project with [MIT License](https://github.com/shaloo/web3-user-personas/blob/main/LICENSE). 
Anyone with reasonable ask and willing can enhance it for the 
benefit of the larger open source community.

## Pre-requisites

* Follow [GitHub Community Guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines).
* Follow [Code of Conduct](https://www.contributor-covenant.org) to interact with the project.

## How to Get Involved?

Follow these guidelines if you wish to collaborate and get involved with the Web3 user personas project.

1. **Feedback:** Share your ideas, input or suggestion via a [GitHub issue](https://github.com/shaloo/web3-user-personas/issues/new). 
2. **Content Update:** If you have some changes or additions that you wish to contribute, follow these steps:

     - Open a new [GitHub issue](https://github.com/shaloo/web3-user-personas/issues/new). 
     - If you are not sure of the updates, first discuss your changes and suggestions with the 
     project team via the [GitHub issue](https://github.com/shaloo/web3-user-personas/issues/new). 
     - If you have clarity on what updates should be made, then in parallel to opening the issue, [fork the repository](https://github.com/shaloo/web3-user-personas/fork).
     - Create a pull request with the title `Fix: #[issue number] phrase describing the changes`.
     - Wait for review comments / merge by the team. 
     - Before submitting a PR, make sure you have verified the changes. 
     - Use the latest sources and submit the PR to the `dev` branch.
     - Address review suggestions if any, use latest source code and issue an updated PR for merging.
