# Web3 User Personas

Web3 user personas are unique and very different from those in Web2. This is largely due
to the following inherent differences in how Web3 operates:

* [[web2-vs-web3#data-ownership|Data ownership]]
* [[web2-vs-web3#identity|User Identity]]
* [[web2-vs-web3#economics|Economics]]
* [[web2-vs-web3#interactions|User Interactions]]
* [[web2-vs-web3#privacy-transparency-trust|Privacy, Transparency, Trust]]

The Web3 persona archtypes can be categorized into five primary types.

* Traders
* Fun & Entertainment
* Builders
* Governance
* Newcomers

To explore details, click on each persona in the table below.
Note that your target persona may be a combination of one or more of these archtypes.
