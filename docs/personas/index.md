# Web3 User Personas

Web3 user personas are unique and very different from those in Web2. This is largely due
to the following inherent differences in how Web3 operates:

* Data ownership
* User Identity
* Economics
* User Interactions
* Privacy, Transparency, Trust
 
[Learn more...](../concepts/web2-vs-web3/)

The Web3 persona archtypes are grouped into four primary types. To explore details, click on the persona.
Note that your target persona may be a combination of one or more of these archtypes.
